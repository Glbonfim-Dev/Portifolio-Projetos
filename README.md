# Portfólio — Guilherme Bonfim

Portfólio profissional desenvolvido para apresentar minha experiência em **Planejamento e Controle da Produção (PCP)**, dados industriais, automação e melhoria de processos.

O site reúne minha trajetória profissional, conhecimentos, projetos desenvolvidos, currículo e formas de contato.

## Funcionalidades

- Apresentação do perfil e da experiência profissional;
- Exibição de conhecimentos em PCP, ERP, Power BI e Excel;
- Carrossel interativo com projetos e apresentações em PDF;
- Download e visualização do currículo;
- Formulário para preparação de contato por e-mail;
- Navegação responsiva para computadores, tablets e celulares;
- Recursos de acessibilidade e suporte à navegação por teclado.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Google Fonts

## Projetos apresentados

- Dashboard de Acompanhamento da Produção;
- Dashboard de Faturamento — Consulta GRV;
- RPA para Consulta e Agrupamento de Dados;
- Agente de IA com dados do SigmaNEST.

## Como visualizar

1. Faça o download ou clone este repositório:

```bash
git clone https://github.com/Glbonfim-Dev/Portifolio.git
```

2. Acesse a pasta `Codigo`.
3. Abra o arquivo `index.html` em seu navegador.

O projeto utiliza apenas tecnologias web nativas, por isso não é necessário instalar dependências.
